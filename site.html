clawd vault

save links to your agent's memory

what

a chrome extension that saves links, selected text, and notes directly to your AI agent's memory. your agent remembers everything you save — no copy-pasting, no bookmarks folder you'll never open again.

backend is openclaw, an open source AI agent platform.

features

  • 1-click save — click the icon, instant save, done
  • right-click any link or page → "save to vault"
  • cmd+shift+v overlay — add notes + pick a category
  • select text on any page, save it with context
  • 7 categories: writing, design, code, ideas, knowledge, edit, film (editable)
  • your agent can search and recall saved links on demand

how it works

the extension talks to a lightweight python server running alongside your openclaw agent. when you save a link, it goes into a local sqlite database and a markdown log your agent reads.

ask your agent "what did i save about X?" and it searches your vault.

install

option 1: tell your agent

send this to your openclaw agent and it handles the rest:

install clawd vault from https://github.com/heettike/clawd-vault

your agent will clone the skill, start the backend, and give you the extension to load.

option 2: manual

# clone the skill git clone https://github.com/heettike/clawd-vault.git cd clawd-vault # install deps + start backend pip install flask flask-cors requests python3 server.py # load extension in chrome # 1. open chrome://extensions # 2. enable developer mode # 3. load unpacked -> select extension/

api

POST /api/save — save a link (url, title, selected_text, notes, category) GET /api/list — recent items (?limit=20) GET /api/search — search saved items (?q=term) GET /api/health — health check

default port: 9877
